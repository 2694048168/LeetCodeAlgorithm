# 命名约定
- 变量名，全部小写，尽量英文单词，做到见名知意，单词之间使用下划线分隔

- 函数名，首字母大写，尽量英文单词，单词之间使用下划线分隔，小驼峰命名法

- 类名，首字母大写，尽量英文单词，单词之间不需要使用下划线分隔，大驼峰命名法

- 文件名，全部小写，尽量英文单词，做到见名知意，单词之间使用下划线分隔

- 在此中可能使用一些常见的算法的缩写以及约定俗成的写法, 例如，梯度下降算法 GD，不需要的变量 _
